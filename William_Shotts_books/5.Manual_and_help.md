# Работа с командами Linux. Получение справки и исследование команд

---

# Цель работы

Научиться:

- получать справочную информацию о командах;
- использовать документацию Linux;
- искать команды по описанию;
- изучать параметры команд;
- понимать структуру man-страниц.

---

# Теоретическая часть

В Linux существует несколько способов получить информацию о командах:

| Команда   | Назначение                       |
| --------- | -------------------------------- |
| `man`     | Руководство пользователя         |
| `--help`  | Краткая справка                  |
| `info`    | Расширенная документация         |
| `whatis`  | Краткое описание команды         |
| `apropos` | Поиск команд по ключевому слову  |
| `which`   | Поиск исполняемого файла команды |

---

# Задание 1. Изучение команды man

## Требуется

Открыть руководство для команды:

```bash
man ls
```

---

## Практика внутри man

Попробовать:

- перейти на страницу вниз (`Space`)
- перейти назад (`b`)
- выполнить поиск (`/`)
- выйти (`q`)

---

# Решение

```bash
man ls
```

---

# Объяснение

`man` (manual) — встроенная документация Linux.

Практически каждая команда имеет собственную man-страницу.

---

# Задание 2. Изучение структуры man-страницы

## Требуется

Открыть:

```bash
man cp
```

Найти разделы:

```text
NAME
SYNOPSIS
DESCRIPTION
OPTIONS
```

---

# Решение

```bash
man cp
```

---

# Объяснение

## NAME

Название команды и краткое описание.

---

## SYNOPSIS

Синтаксис использования.

Пример:

```text
cp [OPTION]... SOURCE DEST
```

---

## DESCRIPTION

Подробное описание работы команды.

---

## OPTIONS

Список параметров.

---

# Задание 3. Использование --help

## Требуется

Получить краткую справку для команд:

```text
ls
cp
mv
mkdir
```

---

# Решение

```bash
ls --help

cp --help

mv --help

mkdir --help
```

---

# Объяснение

`--help` выводит краткую справку без открытия man.

Удобно при работе на серверах.

---

# Задание 4. Команда whatis

## Требуется

Получить краткое описание команд:

```text
ls
cp
mv
```

---

# Решение

```bash
whatis ls

whatis cp

whatis mv
```

---

# Пример результата

```text
ls (1) - list directory contents
```

---

# Объяснение

Команда показывает описание одной строкой.

Полезно для быстрого ознакомления.

---

# Задание 5. Поиск команд через apropos

## Требуется

Найти команды, связанные с:

```text
copy
directory
password
```

---

# Решение

```bash
apropos copy

apropos directory

apropos password
```

---

# Объяснение

`apropos` ищет команды по ключевым словам в документации.

Аналог поиска по справочной системе Linux.

---

# Задание 6. Поиск расположения команды

## Требуется

Определить расположение команд:

```text
ls
cp
bash
```

---

# Решение

```bash
which ls

which cp

which bash
```

---

# Пример результата

```text
/usr/bin/ls
```

---

# Объяснение

Команда показывает путь к исполняемому файлу.

---

# Задание 8. Практика поиска информации

## Ситуация

Вы забыли как работает команда создания каталогов.

Необходимо самостоятельно найти информацию.

---

# Требуется

Найти:

1. Команду создания каталогов.
2. Параметр для создания вложенных каталогов.
3. Пример использования.

---

# Решение

Поиск:

```bash
apropos directory
```

```bash
man mkdir
```

```bash
/ -p
```

внутри man.
