# Команды и конструкции, которые должны быть освоены

| Конструкция  | Назначение                     |
| ------------ | ------------------------------ |
| `~`          | Домашний каталог               |
| `$HOME`      | Путь к домашнему каталогу      |
| `$USER`      | Имя пользователя               |
| `$PATH`      | Пути поиска команд             |
| `$(команда)` | Подстановка результата команды |
| `*`          | Любое количество символов      |
| `?`          | Один символ                    |
| `[abc]`      | Символ из набора               |
| `[0-9]`      | Диапазон символов              |
| `{a,b,c}`    | Перечисление                   |
| `{1..10}`    | Последовательность значений    |
