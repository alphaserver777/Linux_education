# Что такое файловая система Linux

Linux использует древовидную структуру каталогов.

Вершина дерева:

```text id="j9d4j2"
/
```

называется:

- корневой каталог;
- root directory.

---

# Что такое текущий рабочий каталог

Каталог, в котором пользователь находится в данный момент.

Для просмотра используется команда:

```bash id="c5v71w"
pwd
```

что означает:

```text id="jghu7m"
print working directory
```

---

# Абсолютный путь

Полный путь от корня `/`

Пример:

```text id="x7dy4t"
/usr/bin
```

---

# Относительный путь

Путь относительно текущего каталога.

Пример:

```text id="b2a8zc"
cd bin
```

---

# Специальные обозначения

| Символ | Значение             |
| ------ | -------------------- |
| `.`    | текущий каталог      |
| `..`   | родительский каталог |
| `~`    | домашний каталог     |

---

# Основные команды

| Команда | Назначение               |
| ------- | ------------------------ |
| `pwd`   | Показать текущий каталог |
| `ls`    | Показать содержимое      |
| `cd`    | Сменить каталог          |
