# Что такое командная оболочка (Shell)

Командная оболочка — это программа, которая:

- принимает команды пользователя;
- передаёт их операционной системе;
- выводит результат выполнения.

В большинстве Linux-дистрибутивов используется оболочка:

```text id="x1v5k7"
bash
```

что означает:

```text id="jlwmk7"
Bourne Again Shell
```

---

# Что такое терминал

Терминал (terminal emulator) — это программа, через которую пользователь работает с shell.

Популярные терминалы:

- gnome-terminal
- konsole
- xterm

---

# Приглашение командной строки

После запуска терминала обычно отображается строка вида:

```text id="bb8x6d"
user@linux:~$
```

---

# Что означает каждая часть

| Часть   | Значение             |
| ------- | -------------------- |
| `user`  | имя пользователя     |
| `linux` | имя компьютера       |
| `~`     | домашний каталог     |
| `$`     | обычный пользователь |

---

# Символ `$` и `#`

| Символ | Значение                 |
| ------ | ------------------------ |
| `$`    | обычный пользователь     |
| `#`    | root / суперпользователь |

# Задание 3 — История команд

## Требуется

1. Ввести несколько команд:

    - `date`
    - `cal`
    - `pwd`

2. Использовать:

    - стрелку ↑
    - стрелку ↓

---

# Решение

```bash id="8n1iqg"

cal
pwd
```

После этого:

- нажать ↑
- посмотреть предыдущие команды.

---

# Объяснение

Bash сохраняет историю команд.

Это позволяет:

- быстро повторять команды;
- редактировать старые команды;
- ускорять работу администратора.
