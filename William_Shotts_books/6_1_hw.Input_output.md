# Цель домашней работы

Закрепить навыки:

- работы со стандартными потоками Linux;

1. Все задания выполнять только через терминал.
2. Не использовать текстовые редакторы для создания отчётов.
3. Все файлы должны формироваться командами Linux.
4. После каждого задания проверять результат командой:

```bash

```

или

```bash
less имя_файла
```

# Команды, которые должны быть освоены

| Команда   | Назначение             |
| --------- | ---------------------- |
| `>`       | Перезапись файла       |
| `>>`      | Дописывание в файл     |
| `2>`      | Перенаправление ошибок |
| `2>&1`    | Объединение потоков    |
| `\|`      | Конвейер               |
| `cat`     | Вывод файла            |
| `less`    | Просмотр файла         |
| `whoami`  | Текущий пользователь   |
| `df -h`   | Информация о дисках    |
| `free -h` | Информация о памяти    |
