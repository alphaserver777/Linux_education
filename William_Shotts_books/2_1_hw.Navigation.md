# Команды, которые должны быть освоены

| Команда | Назначение           |
| ------- | -------------------- |
| `pwd`   | Текущий каталог      |
| `cd`    | Смена каталога       |
| `ls`    | Просмотр содержимого |
| `ls -a` | Скрытые файлы        |
| `ls -l` | Подробный список     |
| `cd ..` | На уровень выше      |
| `cd -`  | Предыдущий каталог   |
| `cd ~`  | Домашний каталог     |
| `tree`  | Структура каталогов  |
