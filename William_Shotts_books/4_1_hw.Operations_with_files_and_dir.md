# Команды, которые должны быть освоены

| Команда | Назначение                   |
| ------- | ---------------------------- |
| `pwd`   | Текущий каталог              |
| `cd`    | Переход между каталогами     |
| `ls`    | Просмотр содержимого         |
| `mkdir` | Создание каталогов           |
| `touch` | Создание файлов              |
| `cp`    | Копирование                  |
| `mv`    | Перемещение/переименование   |
| `rm`    | Удаление                     |
| `rmdir` | Удаление пустых каталогов    |
| `ln`    | Создание hard link           |
| `ln -s` | Создание symbolic link       |
| `file`  | Определение типа файла       |
| `tree`  | Просмотр структуры каталогов |
