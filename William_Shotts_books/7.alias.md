# Что такое alias

Alias (псевдоним) — это сокращённое имя для команды.

Например:

Вместо:

```bash
ls -lah
```

можно писать:

```bash
ll
```

если заранее создать псевдоним.

---

# Зачем это нужно DevOps-инженеру

На практике ежедневно выполняются десятки одинаковых команд:

- просмотр логов;
- проверка дисков;
- проверка памяти;
- подключение к серверам;
- просмотр контейнеров;
- просмотр процессов.

Чтобы не вводить длинные команды каждый раз, создают alias.

# Задание 3. Создание alias для системной диагностики

# Задание 7. Временный обход alias

## Ситуация DevOps

На сервере alias:


# Задание 8. Постоянное сохранение alias

## Ситуация DevOps

После перезагрузки alias исчезают.

---

## Требуется

Добавить alias в файл:

```text
~/.bashrc
```

---

# Решение

Открыть:

```bash
nano ~/.bashrc
```

---

Добавить:

```bash
alias ll='ls -lah'
alias mem='free -h'
alias disk='df -h'
```

---

# Применить изменения

```bash
source ~/.bashrc
```

---

# Проверка

```bash
ll
mem
disk
```

---

# Объяснение

Теперь alias будут работать после каждого входа в систему.

# Задание 9. Настройка среды Junior DevOps

## Ситуация

Вы получили новый сервер.

Необходимо настроить базовый набор команд.

---

## Требуется

Создать alias:

```bash

mem
disk
ports
proc
```

---

# Решение

```bash
alias logs='less /var/log/syslog'

alias mem='free -h'

alias disk='df -h'

alias ports='ss -tulnp'

alias proc='ps aux'
```

---

# Проверка

Выполнить каждую команду.

---

# Объяснение

Это уже близко к реальной настройке рабочего окружения администратора.

# Основные команды урока

| Команда            | Назначение                      |
| ------------------ | ------------------------------- |
| `alias`            | Создание и просмотр псевдонимов |
| `unalias`          | Удаление псевдонима             |
| `source ~/.bashrc` | Применение настроек             |
| `free -h`          | Память                          |
| `df -h`            | Диски                           |
| `ps aux`           | Процессы                        |
| `ss -tulnp`        | Порты                           |
| `less`             | Просмотр логов                  |

# Что реально используют DevOps

Очень часто в `.bashrc` можно встретить:

```bash
alias ll='ls -lah'
alias grep='grep --color=auto'
alias df='df -h'
alias free='free -h'
alias ports='ss -tulnp'
alias logs='journalctl -xe'
alias k='kubectl'
alias d='docker'
alias dc='docker compose'
```

Это хороший пример того, как автоматизация начинается даже с простых команд командной строки.
