# Часть 8. Постоянное сохранение настроек

---

# Задание 8

## Требуется

Добавить в файл:

```text

```

следующие alias:

```bash

alias mem='free -h'

```

---

## После этого

Применить настройки:

```bash
source ~/.bashrc
```

Перезапустить терминал и убедиться, что alias работают.
