| --------- | -------------------- |
| `date`    | Дата и время         |
| `cal`     | Календарь            |
| `pwd`     | Текущий каталог      |
| `df`      | Использование дисков |
| `free`    | Использование памяти |
| `history` | История команд       |
| `exit`    | Выход из shell       |
