# Стандартные потоки Linux

Каждая программа работает с тремя потоками:

| Поток  | Номер | Назначение    |
| ------ | ----- | ------------- |
| stdin  | 0     | Ввод          |
| stdout | 1     | Обычный вывод |
| stderr | 2     | Ошибки        |

---

# Перенаправление вывода

Оператор:

```bash
>
```

перенаправляет вывод команды в файл.

Пример:

```bash
ls > files.txt
```

Результат:

```text
То, что раньше выводилось на экран,
теперь записывается в файл.
```

---

# Дописывание в файл

Оператор:

```bash
>>
```

не перезаписывает файл, а добавляет данные в конец.

Пример:

```bash
date >> report.txt
```

---

# Конвейер (Pipe)

Оператор:

```bash
|
```

передаёт вывод одной команды на вход другой.

Пример:

```bash
ls /etc | less
```

# Задание 1. Создание первого отчёта

## Ситуация DevOps

Вам необходимо собрать информацию о домашнем каталоге пользователя и отправить её старшему инженеру.

---

## Требуется

Создать файл:

```text
home_report.txt
```

с содержимым каталога.

---

# Решение

```bash
cd ~

ls > home_report.txt
```

---

# Проверка

```bash
cat home_report.txt
```

---

# Объяснение

Без перенаправления:

```bash
ls
```

выводится на экран.

С перенаправлением:

```bash
ls > home_report.txt
```

сохраняется в файл.

---

# Задание 2. Перезапись файла

## Требуется

1. Создать файл отчёта.
2. Записать туда дату.
3. Затем записать туда содержимое каталога.

---

# Решение

```bash
date > report.txt

ls > report.txt
```

---

# Проверка

```bash
cat report.txt
```

---

# Вопрос

Куда исчезла дата?

---

# Объяснение

Оператор:

```bash
>
```

полностью перезаписывает файл.

Это одна из самых частых ошибок начинающих администраторов.

# Задание 6. Разделение информации

- обычный вывод в один файл;

# Задание 7. Объединение потоков

## Требуется

Записать одновременно:

- stdout
- stderr

в один файл.

---

# Решение

```bash
ls /etc /not_exist > all.log 2>&1
```

---

# Объяснение

Конструкция:

```bash
2>&1
```

означает:

```text
stderr направить туда же,
куда направлен stdout
```

# Основные команды урока

| Команда    | Назначение           |          |
| ---------- | -------------------- | -------- |
| `>`        | Перезаписать файл    |          |
| `>>`       | Дописать в файл      |          |
| `2>`       | Перенаправить ошибки |          |
| `2>&1`     | Объединить потоки    |          |
| `          | `                    | Конвейер |
| `cat`      | Вывести файл         |          |
| `less`     | Просмотр файла       |          |
| `df -h`    | Диски                |          |
| `free -h`  | Память               |          |
| `uname -a` | Информация о системе |          |
