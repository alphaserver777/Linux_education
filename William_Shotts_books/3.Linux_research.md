# Теоретическая часть

---

# Исследование системы Linux

После освоения навигации пользователь начинает:

- изучать структуру Linux;
- анализировать файлы;
- исследовать системные каталоги.

Для этого используются команды:

| Команда | Назначение                 |
| ------- | -------------------------- |
| `ls`    | Просмотр содержимого       |
| `file`  | Определение типа файла     |
| `less`  | Просмотр содержимого файла |

---

# Команда `ls`

## Назначение

Выводит:

- файлы;
- каталоги;
- атрибуты файлов.

---

# Базовый синтаксис

```bash id="l3x1"
ls
```

---

# Просмотр другого каталога

```bash id="l3x2"
ls /usr
```

---

# Подробный вывод

```bash id="l3x3"
ls -l
```

---

# Что показывает `ls -l`

Пример:

```text id="l3x4"
drwxr-xr-x 2 user user 4096 May 27 10:00 Documents
```

---

# Разбор строки

| Часть       | Значение          |
| ----------- | ----------------- |
| `d`         | тип объекта       |
| `rwxr-xr-x` | права доступа     |
| `2`         | количество ссылок |
| `user`      | владелец          |
| `4096`      | размер            |
| `May 27`    | дата              |
| `Documents` | имя               |

---

# Команда `file`

## Назначение

Определяет тип файла.

---

# Пример

```bash id="l3x5"
file /bin/ls
```

---

# Возможный результат

```text id="l3x6"
/bin/ls: ELF 64-bit executable
```

---

# Что умеет `file`

Определять:

- текстовые файлы;
- бинарные файлы;
- изображения;
- архивы;
- скрипты;
- исполняемые файлы.

---

# Команда `less`

## Назначение

Постраничный просмотр файлов.

---

# Пример

```bash id="l3x7"
less /etc/passwd
```

---

# Управление в less

| Клавиша | Действие           |
| ------- | ------------------ |
| `↑ ↓`   | прокрутка          |
| `Space` | следующая страница |
| `b`     | назад              |
| `q`     | выход              |
